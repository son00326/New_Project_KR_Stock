# BACKLOG (A) — 무료 애널리스트 컨센서스 → AI 입력 (출시 후 / S7b 인근 검토)

**Status:** BACKLOG (미착수). 출시 critical path 아님. 비용 0 경로. 사용자가 "나중에 진행할 수 있게 놓치지 않도록" 박제 요청(2026-06-19).
**Owner:** USER 결정 + CLAUDE 구현. **Trigger:** W1 AI 토론 / 30 리포트 작성 레이어가 안정된 후(S7b 인근 또는 출시 후 fast-follow).

---

## 왜 이 아이디어인가 (campaign 결론에서 도출)
Tier0 스코어링 캠페인(2026-06, 전부 Claude↔omxy CONVERGED) 결론: **무료 데이터(가격·거래량·크기·외국인·DART 실적·모멘텀·순주식수·차트지표) 정량 깔때기는 게이트 통과 예측 스킬이 없다.** 남은 *대표적·유력한* 학술 예측 신호 후보 = **애널리스트 추정치 변화(estimate revision)** 인데, 이건 **백테스트용 PIT 이력이 유료**(FnGuide/WiseFn 기업용 연 수백만~천만원대)다 — 3인 내부 도구엔 과투자.

**핵심 통찰:** 애널리스트 추정치의 실질 가치는 *정량 팩터로 백테스트*하는 게 아니라, **사람·AI가 종목을 평가할 때 읽는 질적 맥락**(목표주가 컨센서스, 실적 추정 방향/변화)이다. 그건 **무료 소스로 충분**하고, 백테스트 검증이 불필요하다.

## 제안 (비용 0)
**무료 현재 컨센서스를 W1 AI 토론 + 30 리포트 작성의 *컨텍스트 입력*으로 주입** (정량 Tier0 팩터로 넣지 않는다 — 캠페인이 funnel 주입은 recall만 깎는다고 확증):
- **소스(무료)**: 네이버 증권 종목 페이지 컨센서스 · [한경 컨센서스](https://consensus.hankyung.com) (애널 리포트 + 목표주가/추정치 aggregation).
- **주입 지점**: Tier1 Core-11 토론 프롬프트 prior + Section 작성 시 "컨센서스 목표주가/추정 방향" 컨텍스트.
- **DART와의 차이**: DART = 제출된 *실적*(과거). 컨센서스 = *추정*(미래 방향) — 보완재.

## 한계 / 주의 (착수 시 재확인)
- **ToS/스크래핑**: 네이버/한경 자동 수집은 ToS·robots 확인 필요. 소량·캐시·예의 있는 fetch.
- **PIT 이력 없음**: 무료 소스는 *현재* 컨센서스만 — 과거 시점 백필 불가. 백테스트하려면 (가) 유료 PIT 구매 또는 (나) 지금부터 일일 스냅샷 축적(forward-only, 6~12개월 후 검증). **그러나 본 제안은 백테스트가 아니라 AI 질적 입력이므로 PIT 불요.**
- **claim 경계**: 컨센서스는 AI의 *판단 보강* 용도. "사세요/오릅니다" claim 금지(BusinessPlan §7) 유지. 멤버 공개(Deferred-D) 시 추가 검토.
