### 1.2 Summary (1페이지, 맨 앞 고정) — 확정

| 항목 | 포함 내용 |
|------|----------|
| 기본 정보 | 현재가, 52주 범위, 시가총액, 컨센서스 목표가(TP) |
| 핵심 밸류에이션 | 해당 섹터에서 가장 의미 있는 지표 2~3개 (예: 바이오는 Forward PER/PSR, 반도체는 PBR/EV/EBITDA) |
| 기술적 스냅샷 | RSI, 주요 이동평균선 대비 위치, YTD 수익률, 연간 변동성, 52주 MDD(최대낙폭) |
| 투심위 최종 의견 | 섹터 보드 → Core Committee → 종합 투표 테이블 |
| 시나리오별 TP | Bull/Base/Bear + 확률 + 확률가중 TP |

**작성 원칙**:
- 이 페이지만 읽어도 "BUY/HOLD/SELL 중 무엇인지, 왜인지, 리스크는 무엇인지" 파악 가능해야 함
- 숫자 위주, 서술 최소화
- 모든 약어는 괄호 풀이 포함 (예: MDD(최대낙폭), TP(목표가))

### 1.3 Section 0: 투자 서사 (Investment Narrative) — 확정

> **이 섹션에서 알 수 있는 것**: 이 회사가 무엇을 하고, 왜 매력적이며, 왜 지금 관심을 가져야 하는지.

**목적**: 비전문가가 산업 지식 없이도 "이 회사에 왜 관심을 가져야 하는지" 3분 안에 이해할 수 있도록 설명.

**필수 구성 요소**:

| 구성 | 내용 | 분량 |
|------|------|------|
| "이 회사는 무엇을 하는가" | 핵심 사업을 일상 언어로 설명. 전문용어 사용 금지. | 3~4문장 |
| "왜 매력적인가" | 투자 핵심 논리(thesis). 정량 근거 1~2개 포함. | 3~4문장 |
| "왜 지금인가" | 타이밍 논리. 최근 이벤트, 밸류에이션 변화, 시장 환경 등. | 2~3문장 |
| 핵심 수치 카드 | 매출 성장률, 시총, 시장 점유율 등 논거를 뒷받침하는 숫자 3~5개 | 카드 형태 |

**작성 원칙**:
- **비유 필수**: 사업 모델을 일상생활의 비유로 최소 1회 설명
  - 예) "알테오젠은 주사약의 배달 방식을 바꾸는 회사입니다. 병원에서 1시간 걸리던 링거(정맥주사, IV)를 5분짜리 피하주사(SC)로 바꿔주는 기술 플랫폼을 가지고 있습니다."
  - 예) "삼성전자의 반도체 사업은 '디지털 부동산 임대업'에 가깝습니다. 데이터 저장 공간(메모리)을 만들어 빅테크에 대량 공급합니다."
- "이 회사에 투자해야 합니다" 같은 추천 문구 사용 금지 (법적 제약). "~한 이유로 주목할 만합니다" 수준.
- 후속 섹션의 핵심 논거를 미리 훑되, 상세는 해당 섹션에서 다룸을 명시

### 1.4 Section 1: 산업 기초 (Industry Fundamentals) — 확정

> **이 섹션에서 알 수 있는 것**: 이 회사가 속한 산업의 구조, 규모, 경쟁 환경.

**목적**: 독자가 해당 산업을 처음 접하더라도 핵심 역학을 이해할 수 있도록 기초부터 설명.

**필수 구성 요소**:

| 구성 | 내용 |
|------|------|
| 산업 개요 | 이 산업이 무엇인지 비전문가에게 설명 (비유 활용). 산업의 핵심 동인(driver) 3가지. |
| TAM/SAM 산출 | TAM(전체시장규모)과 SAM(접근가능시장규모)을 구체적 근거와 함께 산출. "~조원 시장"이라고만 쓰지 말고 산출 과정을 보여줄 것. |
| 경쟁 포지션 | 해당 산업에 적합한 비교 축으로 경쟁사 비교 (§4 비교 분석 가이드라인 참조). 경쟁사 최소 3개. |
| 산업 구조 | 밸류체인 또는 산업 생태계 도식화 (텍스트 다이어그램 권장). |
| 업황 사이클 | 현재 산업이 사이클 어디에 있는지 (상승/정점/하강/바닥). 판단 근거 포함. |

**섹터별 특수 사항 (예시)**:
- 바이오: 파이프라인 성공 확률 체계, 규제 경로(FDA/EMA/MFDS), 특허 클리프
- 반도체: 메모리 사이클, 공정 노드 경쟁, 고객 집중도
- 건설: 수주잔고 사이클, 금리 민감도, PF 리스크
- 금융: NIM(순이자마진) 추이, 건전성 지표, 규제 자본

### 1.5 Section 2: 회사 개요 (Company Overview) — 확정

> **이 섹션에서 알 수 있는 것**: 이 회사의 역사, 경영진, 지분 구조, 재무 안정성, 핵심 기술.

**필수 구성 요소**:

| 구성 | 내용 |
|------|------|
| 히스토리 타임라인 | 설립부터 현재까지 핵심 이벤트 (연도별 5~10개). 테이블 형태 권장. |
| **성장 배경 추적 (Track Record)** | 회사가 과거에 제시한 계획(가이던스, 목표, 로드맵) vs 실제 이행 결과 비교. **"말한 대로 하는 회사인가"** 판단의 핵심. 최소 3개년 가이던스 vs 실적 테이블 포함. 미이행 항목은 사유 분석. |
| 경영진 | 핵심 인물 3~5명. 이름, 직함, 경력 하이라이트, 역량 평가(강점/약점). |
| 지분 구조 | 대주주/내부자 지분율 + 최근 변동. 오버행(대량 매도 가능성) 여부. |
| 재무 안정성 | 현금 보유량, 부채비율, 유동비율. "자금이 마를 위험이 있는가" 한 줄 판단. |
| 핵심 기술/자산 | 비전문가를 위한 기술 설명. **비유 필수**. "이 기술이 왜 가치 있는지"를 일상 언어로. |

**작성 원칙**:
- **성장 배경 추적은 투자 판단의 핵심 근거**. 가이던스 달성률이 높은 회사 = 경영진 신뢰도 높음 = 밸류에이션 프리미엄 정당화. 미달성 시 원인(외부 환경 vs 실행 역량)을 구분하여 기술.
  - 예시 (바이오): "2023년 '2건 추가 딜' 목표 → 실제 3건 달성. 2024년 '매출 1,000억' 가이던스 → 1,029억 달성(+2.9%). 경영진 가이던스 신뢰도: 높음."
  - 예시 (반도체): "2024년 'HBM 점유율 30%' 목표 → 실제 17%에 그침. 수율 문제로 미달. 2025년 35%로 재제시 → 달성 여부 핵심 모니터링 포인트."
- 경영진 평가는 사실 기반으로만 (경력, 실적, 의사결정 이력). 주관적 인물평 지양.
- 지분 구조 변동은 최근 1년 기준. 내부자 매도가 있었다면 맥락(세금, 이혼, 유동성 등) 포함.
- 핵심 기술 설명에서 **"마치 ~와 같다"** 형태의 비유를 반드시 1회 이상 사용.
  - 예) "ALT-B4는 '분자 택배 서비스'입니다. 약물(택배)을 피하(SC, 피부 아래)로 빠르게 전달하기 위해 히알루론산(포장재)을 분해하는 효소를 사용합니다."

### 1.6 Section 3: 이익 모델 (Earnings Model) — 확정

> **이 섹션에서 알 수 있는 것**: 이 회사가 어떻게 돈을 버는지, 앞으로 얼마나 벌 것인지.

**필수 구성 요소**:

| 구성 | 내용 |
|------|------|
| 수익 구조 비유 | 돈 버는 방식을 비유로 먼저 설명 후 상세 전개. |
| 과거 실적 | 최근 3개년 매출/영업이익/순이익/OPM(영업이익률). 테이블 필수. |
| 당해 컨센서스 | 증권사 컨센서스 평균 + 분포(최고/최저). 출처 명시. |
| 중장기 매출 로드맵 | 3~5년 매출 성장 시나리오. 핵심 드라이버별 기여도 분해. |
| 핵심 드라이버 분류 | **실현(확정)** vs **미실현(기대)** 명확 분리. 미실현 드라이버에는 실현 확률 표기. |

**작성 원칙**:
- 수익 구조 비유 예시:
  - 바이오: "알테오젠의 수익은 '기술 라이선싱 로열티'입니다. 좋은 레시피를 여러 레스토랑에 빌려주고, 계약금(업프론트) + 성과급(마일스톤) + 매출 수수료(로열티)를 받는 구조."
  - 반도체: "삼성 반도체의 수익 구조는 '메모리 시세'에 달려 있습니다. 메모리 가격이 오르면 같은 양을 팔아도 이익이 급등하고, 떨어지면 급감하는 '원자재형' 수익."
- 마일스톤, 일회성 수익 등은 반드시 분리 표기. Trailing 지표의 왜곡 가능성 경고 포함.

### 1.7 Section 4: 밸류에이션 (Valuation) — 확정

> **이 섹션에서 알 수 있는 것**: 이 회사의 현재 주가가 비싼지 싼지.

**필수 구성 요소**:

| 구성 | 내용 |
|------|------|
| Peer 비교 | 동종 peer 최소 3개 이상. 비교 축은 섹터에 적합한 지표 사용. 테이블 필수. |
| Multiple 분석 | 해당 섹터에서 의미 있는 배수 지표. 각 지표에 "이 숫자가 의미하는 것" 한 줄 해석 필수. |
| DCF (해당 시) | Sensitivity table (WACC × 성장률) 포함. 주요 가정 명시. |
| 증권사 TP 분포 | 컨센서스 평균, 최고, 최저. 괴리율. |
| "비싼가 싼가" 결론 | 벤치마크 대비 명확한 판단 한 줄. |

**밸류에이션 지표별 한 줄 해석 가이드**:

| 지표 | 한 줄 해석 템플릿 |
|------|------------------|
| PER (주가수익비율) | "현재 이익 기준 투자금 회수에 X년 걸린다는 뜻. peer 평균 Y배 대비 [높다/낮다]." |
| Forward PER | "내년 예상 이익 기준 X배. 성장이 반영되면 [합리적/과도]." |
| PBR (주가순자산비율) | "장부가 대비 X배. 1 미만이면 청산가치 이하로 거래되는 것." |
| PSR (주가매출비율) | "매출 대비 X배. 아직 이익이 없는 성장주에 적합한 지표." |
| EV/EBITDA | "기업 전체 가치(부채 포함) 대비 영업현금흐름. 인수 매력도 판단에 유용." |
| ROE (자기자본이익률) | "주주 돈 100원으로 X원을 벌었다는 뜻. 15% 이상이면 한국 평균 대비 우수." |

**작성 원칙**:
- "삼성전자 대비" 같은 크로스섹터 비교 금지. 동일 산업 peer만 사용.
- DCF는 모든 섹터에 강제가 아님 (바이오텍처럼 FCF 변동이 극심한 경우 "DCF 한계" 명시 후 대체 방법론 사용).
- Sensitivity table에는 최소 3×3 매트릭스 (WACC 3단계 × 성장률 3단계).

### 1.8 Section 5: 거시 민감도 (Macro Sensitivity) — 확정

> **이 섹션에서 알 수 있는 것**: 금리, 환율, 원자재 등 거시 환경이 이 회사에 미치는 영향.

**필수 구성 요소**:

| 구성 | 내용 |
|------|------|
| 매크로 민감도 매트릭스 | 주요 거시 변수(금리/환율/원자재/정책 등) × 영향도(상/중/하) × 방향(양/음). |
| 현재 매크로 환경 판단 | 현재 각 변수의 방향성 + 이 종목에 우호적인지 비우호적인지. |
| 섹터 자금 흐름 | 외국인/기관 순매수 추이. 해당 섹터에 자금이 유입/이탈 중인지. |

**작성 원칙**:
- "금리가 오르면 불리합니다" 같은 일반론 금지. **구체적 전달 경로** 명시.
  - 예) "금리 +50bp → 바이오 할인율 상승 → DCF TP -8% → 성장주 밸류에이션 압박"
- 환율 영향은 실적(달러 수익) + 밸류에이션(외국인 투자) 양쪽 분리 기술.

| **동일 섹터 peer만** | 크로스섹터 비교 금지. "삼성전자 대비 알테오젠" 같은 비교는 무의미. |
| **최소 3개 peer** | 1~2개로는 통계적 의미 없음. 가능하면 5개 이상. |
| **국내 + 해외 모두 필수** | 국내 peer만으로는 글로벌 포지셔닝을 판단할 수 없다. 반드시 국내 2~3개 + 해외 2~3개 이상 포함. 환율 조정 명시. |
| **비교 축은 섹터별** | 모든 섹터에 PER만 쓰지 않음. 아래 섹터별 가이드 참조. |
| **딜/계약 조건 상대 비교** | 해당 산업에서 딜밸류, 로열티율, 계약 규모 등이 중요한 경우, 경쟁사의 유사 딜과 조건을 비교하여 "이 딜이 유의미한 수준인지" 판단. 업계 평균/범위 제시 필수. |
| **시총 구간 유사** | 대형주와 소형주를 같은 테이블에 넣을 때는 시총 차이를 명시. |

### 5.2 독립성 원칙 (확정)

1. **동일 논거 2명 이상 사용 금지**: 같은 결론(BUY)이라도 근거가 달라야 함. "플랫폼 가치"를 논거로 쓴 위원이 있으면, 다른 위원은 다른 각도(밸류에이션, 기술적 분석, 매크로 등)에서 접근.
2. **투자 철학 반영 강제**: Core Committee 각 위원은 자신의 투자 철학에서 자연스럽게 나오는 관점만 사용. Buffett이 기술적 분석을 논거로 쓰거나, Druckenmiller가 "안전마진"을 주장하면 안 됨.
3. **고유 통찰 의무**: 섹터 보드 위원은 자신의 전문 분야에서만 나올 수 있는 통찰을 반드시 1개 이상 포함.

### 5.4 면책 문구 (확정, 모든 보고서 필수 포함)
