| **리스크** | 특허 분쟁, 임상 실패, 오버행, 물질특허 만료, 규제 변화 | 뉴스, 법원 공시, USPTO/PTAB |
| **촉매** | 향후 6~12개월 이벤트 캘린더 | IR, 학회 일정, FDA PDUFA |

## 바이오 특수 밸류에이션 주의사항

1. **Trailing PER은 무의미**: 바이오텍은 마일스톤 일시인식으로 Trailing PER 왜곡. **Forward PER** 또는 **PSR** 기준 판단
2. **DCF의 한계**: 바이오텍 FCF는 딜 타이밍에 따라 극단적 변동. "정상화 FCF" 추정 시 마일스톤 제외 근거 명시
3. **로열티 확정 vs 추정 구분**: SEC Filing에서 확인된 로열티만 "확정". 나머지는 반드시 [추정] 표기
4. **파이프라인 가치(rNPV)**: 임상 단계별 성공 확률 × 예상 매출 × 로열티율로 산출. 간이 시나리오 대체 가능

## 향후 추가 예정 데이터 소스 (API)

현재 pykrx(가격 데이터)만 사용 중. 아래 API가 연동되면 보고서 정밀도 대폭 향상:

| API | 제공 데이터 | 보고서 반영 섹션 |
|-----|-----------|----------------|
| **한투 OpenAPI** | 실시간 호가, 외국인/기관 수급, 공매도 | S7 리스크 (수급 정량화) |
| **DART OpenAPI** | 공시 원문, 재무제표, 대주주 변동 | S1 실적, 리스크 (오버행) |
| **FDA API (openFDA)** | 허가 이력, PDUFA 일정, Warning Letter | S3 산업, S6 촉매 |
| **ClinicalTrials.gov API** | 임상시험 등록/진행/결과 | S3 파이프라인, S6 촉매 |
| **SEC EDGAR API** | 파트너사 Filing (10-K/10-Q 로열티 공개) | S1 로열티 확정 |
| **PubMed API** | 학술 논문 (기술 검증) | S3 기술 차별점 |
| **뉴스 API (네이버/구글)** | 실시간 뉴스 감성 분석 | S4 섹터 심리, S6 촉매 |
| **Bloomberg/Refinitiv** | 글로벌 피어 밸류에이션, 컨센서스 | S2 밸류에이션, S5 시나리오 |
